# การจำแนกพื้นที่เขาหัวโล้นด้วย Random Forest จากข้อมูล Sentinel-2

โครงการนี้เป็นการพัฒนาแบบจำลอง **Random Forest (RF)** สำหรับจำแนกพื้นที่เขาหัวโล้น (Bare Mountain) ในจังหวัดน่าน โดยใช้ข้อมูลภาพถ่ายดาวเทียม Sentinel-2 ร่วมกับดัชนีเชิงสเปกตรัมและข้อมูลภูมิประเทศ เพื่อจัดทำแผนที่พื้นที่เขาหัวโล้นและประเมินความถูกต้องของแบบจำลอง

---

## วัตถุประสงค์

- จำแนกพื้นที่เขาหัวโล้นจากภาพ Sentinel-2 ปี 2567
- พัฒนาแบบจำลอง Random Forest สำหรับการจำแนกข้อมูล
- ประเมินความถูกต้องของแบบจำลอง
- คำนวณพื้นที่เขาหัวโล้น
- ตรวจสอบผลลัพธ์ด้วยการสุ่มตรวจ (Visual Validation)

---

## พื้นที่ศึกษา

จังหวัดน่าน ประเทศไทย

ระบบพิกัด

```
WGS84 / UTM Zone 47N (EPSG:32647)
```

---

## ข้อมูลที่ใช้

## Sentinel-2 Level-2A (2024)

ใช้แบนด์

- B2 (Blue)
- B3 (Green)
- B4 (Red)
- B8 (NIR)
- B11 (SWIR1)
- B12 (SWIR2)

---

## ดัชนีที่ใช้

- NDVI
- NDMI
- NBR
- BSI
- EVI

---

## ข้อมูลภูมิประเทศ

- Elevation
- Slope
- Eastness
- Northness

รวมทั้งหมด **15 ตัวแปร**
