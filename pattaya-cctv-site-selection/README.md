# การวิเคราะห์ตำแหน่งติดตั้งกล้องวงจรปิด (CCTV) ที่เหมาะสมในเมืองพัทยาด้วยเทคนิค GIS และ Multi-Criteria Decision Analysis

## รายละเอียดโครงการ

โครงการนี้เป็นการพัฒนาแบบจำลองเชิงพื้นที่ (Spatial Model) สำหรับคัดเลือกตำแหน่งติดตั้งกล้องวงจรปิด (CCTV) ที่เหมาะสมในเขตเมืองพัทยา โดยประยุกต์ใช้ระบบสารสนเทศภูมิศาสตร์ (Geographic Information System: GIS) ร่วมกับการวิเคราะห์หลายเกณฑ์ (Multi-Criteria Decision Analysis: MCDA)

วัตถุประสงค์ของโครงการคือการวิเคราะห์พื้นที่ที่มีความเสี่ยงจากหลายปัจจัย พร้อมจัดลำดับความสำคัญของตำแหน่งติดตั้งกล้องวงจรปิด เพื่อสนับสนุนการวางแผนด้านความปลอดภัยและการบริหารงบประมาณของเมือง

---

## วัตถุประสงค์

- วิเคราะห์พื้นที่เสี่ยงด้านความปลอดภัยของเมืองพัทยา
- พัฒนาแบบจำลองเชิงพื้นที่สำหรับคัดเลือกตำแหน่งติดตั้ง CCTV
- วิเคราะห์ความเสี่ยงจากหลายปัจจัยด้วย MCDA
- จัดลำดับความสำคัญของจุดติดตั้งกล้องวงจรปิด

---

## ข้อมูลที่ใช้

- โครงข่ายถนน (OpenStreetMap)
- ภาพถ่ายแสงสว่างเวลากลางคืน (SDGSAT-1 Nighttime Light)
- ตำแหน่งกล้องวงจรปิดเดิมของเมืองพัทยา
- ตำแหน่งสถานีตำรวจ
- ข้อมูลความหนาแน่นประชากร
- ข้อมูลแหล่งท่องเที่ยว
- ข้อมูลจุดเกิดเหตุอาชญากรรม
- ข้อมูลการใช้งานพื้นที่ (Footfall Density)

---

## วิธีการศึกษา

โครงการแบ่งการวิเคราะห์ออกเป็น 5 แบบจำลอง

### Scenario 1 : Darkness Risk Model

วิเคราะห์พื้นที่เสี่ยงจาก

- ความมืด (Darkness)
- ความหนาแน่นการใช้งานพื้นที่ (Footfall Density)

---

### Scenario 2 : Road Security Risk Model

วิเคราะห์พื้นที่ที่มีความสำคัญต่อการสัญจรของคนเดินเท้า โดยพิจารณา

- ประเภททางเดิน (Pedestrian Weight)
- ความหนาแน่นการใช้งานพื้นที่
- ความมืด
- การเข้าถึงสถานีตำรวจ

---

### Scenario 3 : Tourism Security Model

วิเคราะห์พื้นที่ท่องเที่ยวที่ควรได้รับการเฝ้าระวังเพิ่มเติมจาก

- ความใกล้แหล่งท่องเที่ยว
- ความมืด
- ช่องว่างการครอบคลุมของกล้องวงจรปิดเดิม (CCTV Gap)

---

### Scenario 4 : Crime Hotspot Model

วิเคราะห์พื้นที่เสี่ยงจาก

- ข้อมูลจุดเกิดเหตุอาชญากรรม
- ความมืด
- ความหนาแน่นการใช้งานพื้นที่
- การเข้าถึงสถานีตำรวจ

---

### Scenario 5 : Investment Priority Model

บูรณาการผลลัพธ์จากทุกแบบจำลอง เพื่อจัดลำดับความสำคัญของการลงทุนติดตั้งกล้องวงจรปิด

---

## ขั้นตอนการดำเนินงาน

1. เตรียมข้อมูลเชิงพื้นที่
2. ประมวลผลข้อมูลด้วย GIS
3. Normalize ตัวแปรทั้งหมดให้อยู่ในช่วง 0–1
4. คำนวณคะแนนความเสี่ยงของแต่ละแบบจำลอง
5. จัดระดับความเสี่ยงของถนน
6. สร้าง Candidate Point จากโครงข่ายถนน
7. คัดเลือกจุดติดตั้งโดยกำหนดระยะห่างขั้นต่ำ 100 เมตร
8. จัดอันดับตำแหน่งติดตั้งเป็น Top 50, Top 100 และ Top 250

## ผลลัพธ์

- แผนที่ความเสี่ยงทั้ง 5 แบบจำลอง
- คะแนนความเสี่ยงของถนนแต่ละเส้น
- ตำแหน่งติดตั้งกล้องวงจรปิดที่แนะนำ
- ไฟล์จัดอันดับตำแหน่งติดตั้ง (Excel)
- แผนที่แสดงตำแหน่ง CCTV ที่เสนอ
